Criterios de búsqueda

Guía taxativa para buscar memorandos dentro de Memo Search. Use estos criterios para obtener resultados más precisos y evitar búsquedas demasiado amplias.

Ir a Buscar memorandos Volver al inicio

1. Por número

Use el número cuando lo conozca.

  • Ejemplo: 1471
  • Ejemplo: 920
  • Evite agregar palabras innecesarias si solo busca un número específico.

2. Por dependencia o lugar

Use siglas, zonas, unidades o dependencias.

  • Ejemplo: CABA
  • Ejemplo: PI-7
  • Combine dependencia + tema para mejorar precisión.

3. Por palabra clave

Busque términos concretos.

  • Ejemplo: licencia médica
  • Ejemplo: daño
  • Ejemplo: expediente

4. Por persona

Use apellido, nombre, DNI o legajo.

  • Ejemplo: Pérez
  • Ejemplo: 12345678
  • Pruebe apellido solo y luego apellido + dependencia.

5. Por expediente o archivo

Use parte del nombre del archivo o identificadores.

  • Ejemplo: EX-2024
  • Ejemplo: memorando 1471
  • Ejemplo: 920.pdf

6. Combinación recomendada

La mejor búsqueda combina dato + tema + dependencia.

  • Ejemplo: 1471 daño CABA
  • Ejemplo: PI-7 licencia médica
  • Ejemplo: expediente daño
